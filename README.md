# QR Code Generator Web App

## 📌 Overview
This is a simple web application that generates QR codes from user input (text or URL).

## 🚀 Features
- Generate QR codes instantly
- Download QR code as PNG
- Simple and clean UI

## 🛠 Tech Stack
- Python (Flask)
- qrcode library
- HTML/CSS

## ▶️ How to Run

1. Clone the repo:
git clone https://github.com/your-username/qr-code-generator-web-app.git

2. Install dependencies:
pip install -r requirements.txt

3. Run the app:
python app.py

4. Open browser:
http://127.0.0.1:5000/
